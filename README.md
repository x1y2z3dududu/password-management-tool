# 密码管理器safe (Misuo)
> 一个基于微信小程序的本地加密密码管理工具。所有凭证数据在设备端使用 AES-256 加密存储，主密码永不离开本地。

## 功能介绍
### 凭证管理
- **增删改查**：记录账号密码，支持标题、用户名、密码、网址、备注、分类
- **一键复制**：点击即可复制用户名 / 密码到剪贴板，无需手动选中
- **密码显隐**：默认隐藏密码，点击眼睛图标切换显示
- **密码生成**：内置强密码生成器，可自定义长度和字符类型
### 全局搜索
- **多字段搜索**：支持按标题、用户名、域名、备注、分类搜索
- **智能匹配**：精确匹配 > 前缀匹配 > 包含匹配，结果按相关度排序
- **中文分词**：对中文标题按字符拆分索引，输入任意字即可命中
- **倒排索引**：内存级倒排索引，百条凭证秒级搜索
### 分类标签
- **内置 7 大分类**：社交、邮箱、金融、工作、娱乐、购物、其他
- **自定义分类**：用户可新增、编辑、删除分类，自定义图标和颜色
- **智能迁移**：删除分类时，该分类下的凭证自动归入「其他」，不会丢失
### 跨设备云同步
- **微信云开发**：基于微信 CloudBase，登录即同步，无需额外注册
- **增量合并**：多设备编辑时按 `updatedAt` 自动合并，保留最新版本
- **新设备冷启动**：换手机后打开密锁，自动从云端拉取加密数据
### 安全防护
- **主密码锁定**：首次使用设置主密码，之后每次打开均需验证
- **后台清除**：小程序切入后台瞬间清除内存中的密钥，回到前台必须重新解锁
- **错误次数限制**：连续 5 次密码错误后锁定 30 分钟

## 使用说明
### 普通用户
1. 在微信中搜索「密码管理器safe」小程序或扫描以下二维码
   <img width="2172" height="800" alt="999" src="https://github.com/user-attachments/assets/4edf849f-1c77-483d-aa92-13a42697db68" />
2. 首次打开 → 设置主密码（至少 6 位，建议字母 + 数字 + 符号混合）
3. 点击「+」添加凭证，填写账号信息
4. 需要时点击凭证卡片，一键复制用户名或密码

### 开发者本地运行
```bash
# 1. 克隆仓库
git clone https://github.com/你的用户名/仓库名.git
cd 仓库名

# 2. 安装依赖
npm install

# 3. 用微信开发者工具打开项目目录
#    → 工具 → 构建 npm
#    → 点击「编译」即可在模拟器中运行
```

**前置要求：**
- [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 最新稳定版
- Node.js 16+
- 如需云同步功能，需在微信公众平台开通「云开发」并创建 `user_data` 集合

### 技术栈
| 层面 | 技术 |
|------|------|
| 前端框架 | 微信小程序原生开发（WXML + WXSS + JS） |
| 加密库 | CryptoJS（AES-256-CBC + PBKDF2-SHA256） |
| 云服务 | 微信云开发 CloudBase（数据库 + 用户鉴权） |
